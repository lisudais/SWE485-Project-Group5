## Project Title
Student Depression Risk Analysis Using Machine Learning


## Project Motivation
Student depression is an important issue that can be influenced by several academic and lifestyle factors. Elements such as academic pressure, study workload, sleep patterns, and daily habits may affect students’ mental well-being and increase the risk of depression. Understanding how these factors relate to depression can help identify patterns that indicate higher risk among students.

This dataset was selected because it includes multiple variables describing students’ academic conditions and daily lifestyle habits. These factors provide meaningful information that allows us to analyze how different aspects of students’ lives may influence depression risk and support the development of models that estimate this risk.
